# Our messaging

## Pillars

* **Unified Agentic Discovery (Illuminate)** — Automatically discovers and inventories every API, MCP server, and AI agent across your environment, illuminating shadow and zombie assets without requiring software agents or application code changes.
* **Posture Management & Compliance Governance (AG-SPM)** — Continuously maps live configurations against regulatory frameworks like PCI DSS, GDPR, and the EU AI Act, detecting API drift and misconfigurations before they are exploited.
* **Runtime Behavioral Threat Protection (AG-DR)** — Leverages cloud-scale big data and machine learning to baseline normal behavior over time, identifying and stopping complex business logic abuse and multi-step attacks that bypass traditional edge tools.
* **Salt Code** — Enforces unified security policies directly inside AI coding assistants and developer workflows, ensuring secure APIs and MCP connections are built by default before they ever reach production.

## Positioning language we use

* **Taglines:** "From agent to API. The full agentic lifecycle, secured," "Map your Agentic Security Graph," and "See It, Govern It, and Protect It."
* **Category name we push:** Agentic Security Platform / API Protection Platform.
* **Voice/style rules:** The tone is authoritative, enterprise-grade, and visionary. Messaging strictly differentiates true "security" (threat correlation, action-layer defense) from mere "observability" (model metrics, hallucination monitoring), emphasizing architectural context, big data baselining, and actionable governance.
